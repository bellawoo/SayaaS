# SayaaS

Say as a Service -- SAAS... geddit?

## Commands available

Feelin Faboulous? Try this:
```GET /say_ah```


Don't like Trey Songz? Pick a different voice:
```GET /say_ah/random```


Tell the voices what you weant to hear:
```POST /new_phrase```


Have the voices say what you submitted (maybe, we'll see):
```GET /user_phrase/:id```
